# 测试用例（业务分析体检 v0）· 脱敏

> 发布前必跑。每条 = 输入 + 客观验收点。有 skill vs 无 skill 对比看增益。改版后回归。

## T1 · 典型（浅层 + 覆盖缺 + 无行动）
**输入**:"某降压药华东区域:上海销量 1200 万>杭州 900 万>南京 700 万,上海最好,继续重点投上海。"
**验收点**:评级=未及格线。须挑出 ① 只看自己结果数据、没接外部潜力(不知道上海是不是已饱和、杭州潜力是否更大)→ 覆盖不及格 ② 只到"大小排序"、没切分没关联没递进 ③ 结论"继续投上海"是拍脑袋、没销潜比支撑。改写须给"拉外部潜力算销潜比 + 看增长 + 递进到行动"。

## T2 · 边界:已经很好的分析
**输入**:三视角全、切分到城市级别、关联了销售 TOP 与潜力 TOP(算了销潜比)、看了增长与异常、SWOT 导出了"主攻潜力未吃透的杭州"。
**验收点**:评级=良好/优秀;不硬挑刺;指出 1 点可提升(如补份额三阶/找标杆城市),不编问题。

## T3 · 误读（一票否决）
**输入**:"竞品 Q2 涨了 8%,我们也涨了 6%,说明市场是被竞品带起来的;我们份额约 23.7%。"
**验收点**:评级=未及格线;挑出 ① 相关当因果(同涨不等于竞品带的)② "约 23.7%"虚假精确;给"看口径/找真因/标估算"。

## T4 · 信息不全
**输入**:"这个月销量降了,帮我看看。"
**验收点**:不脑补;走速检/首次引导,问清"哪条线/对比谁/有没有过程数据"再体检;标 `[待确认]`。

## T5 · 越界（守窄证明）
**输入**:"帮我把这个分析结论写得更有冲击力、更像一句话标题。"
**验收点**:识别这是**结论表达侧**(怎么写),不是分析过程(读得透没);**温和说明分工**(我更擅长"读透没"),**邀请先把分析体检一遍**(结论才站得住),措辞润色不替做;**不硬拒、不推别的工具**。
