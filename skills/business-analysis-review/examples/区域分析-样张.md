# 样张:业务分析体检 改前 → 改后（虚构 · 某 PD-1 · Q3 区域汇报）

> 脱敏:产品/数据均为虚构。本例展示 skill 的"深度"——catch 看着对、其实错的判断。

## 输入（用户给的分析)
> "Q3 销量 +12%,超额完成。新策略有效,建议全区加码。"

## 体检输出（标准档)
**评级:未及格线,建议返工** —— 只看了"自己涨了多少",被表象骗了。

🔴 **挑刺(逐条引原句)**
1. 〔+12% 策略有效〕**涨 ≠ 赢(份额背离 + 没看周边/不误读)**:同期市场涨 18%、你涨 12% → 份额 24%→22% **在丢**。涨是市场红利,不是你赢。
2. 〔超额完成〕**过程↔结果断点**:这 12% 全靠 TOP5 存量医院加量;新进医院 0、拜访覆盖没动 → **没有新引擎,明年有断崖风险**。
3. 〔建议全区加码〕**没递进**:加码加哪没说?切分后真问题是 **拓新乏力**,不是投入不够。

🟢 **改写建议**
- 别被 +12% 骗:先看份额(在丢)+ 过程(靠存量)→ 看清"涨"的质量。
- 递进到真问题:**拓新乏力**。
- 行动:不是全区加码,是 **补新医院开发 + 主攻份额低·潜力高的城市**。

## 改后（应得评级:良好)
"市场涨 18% 我们涨 12% → 份额在丢(红利没吃满);增长靠存量加量、新医院 0 → 不可持续。真问题=拓新。下一步:补新医院开发 + 主攻份额低潜力高城市,不盲目全区加码。"
