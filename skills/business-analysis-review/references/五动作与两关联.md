# 五个习惯动作（招牌） + 两个关联

## ★5 个习惯动作（读每一组数据都过一遍）
**看大小 · 看趋势 · 看变化Δ（时间轴）· 看非常规（异常）· 找标杆**
- 配套铁律:**先总后分;给出描述性结论后必须"问原因"。**
- 体检:逐个核——漏了哪个动作?(尤其"看非常规/找标杆"最常漏)

| 动作 | 在看什么 | 漏了会怎样 |
|---|---|---|
| 看大小 | 排序、占比、谁重要 | 抓不住重点 |
| 看趋势 | 涨还是跌、走向 | 静态看不出势 |
| 看变化Δ | 时间轴上的增量/拐点 | 错过拐点 |
| 看非常规 | 异常值、不合常理处 | 漏掉问题/机会信号 |
| 找标杆 | 跟最佳/平均比 | 不知道好坏的尺 |

## 两个关联（最容易漏,是从"描述"到"洞察"的桥）
- **内部 ↔ 外部**:自家数据 vs 外部潜力/竞品。典型交叉 = **销潜比 = 内部销量 ÷ 外部潜力**(看错配:卖得多但潜力更大 = 没吃透)。
- **过程 ↔ 结果**:过程数据(覆盖/拜访/声量)vs 结果数据(销量/份额)→ 找断点(过程好但结果差 = 哪个环节漏了)。

## 挑刺判定
- 五动作没做全 / 描述完不问原因 = 读得不透。
- 只看结果不看过程、只看内部不接外部 = 关联缺失,通常一票否决到"未及格线"。
