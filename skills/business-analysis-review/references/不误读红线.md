# 不误读红线（贯穿全程）

数据可以读浅,但不能读错。命中即一票否决到"未及格线"。

1. **相关 ≠ 因果**:两个数一起涨/跌,不等于一个导致另一个。要追"为什么",别脑补因果。
2. **虚假精确**:样本/口径撑不起的小数位别写。"约 5%" 不要写成 "4.73%";估算值标"估算"。
3. **数据要有出处**:每个关键数标来源/口径/时间;缺则标 `[待确认]`,不裸奔下结论。
4. **口径一致**:同分母、同时间、同范围才比;换口径要说明。
5. **样本偏差**:小样本/非随机别当全局;CPA/小样本数据库注意覆盖偏差。

> 体检时:逐条引原句,指出"这里把相关当因果了 / 这个精度撑不住 / 这个数没出处"。
