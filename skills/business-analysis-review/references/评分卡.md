# 评分卡:一票否决 + 分层判定

## 一票否决（命中任意一条 = 未及格线,建议返工）
- 把相关当因果(如"竞品涨我也涨,所以是竞品带的")
- 数据无出处 / 虚假精确("约 5%" 写成 "4.73%")
- 只单一描述结果数据(谁大谁小),没接外部、没递进

## 分层判定（及格 / 良好 / 优秀）
| 维度 | 未及格线 | 良好 | 优秀 |
|---|---|---|---|
| 工序 | 只解读 | + 切割 + 五动作做全 | + 关联 + 递进到行动启示 |
| 覆盖 | 只内部结果 | + 过程数据 | + 外部潜力/竞品(内外交叉出销潜比) |
| 视角 | 只看自己 | + 看周边 | + 看全局 |
| 指标 | 单一指标 | 份额+潜力+增长三维 | + 漏斗/份额三阶/销潜比 |
| 落地 | 有结论 | 结论接策略 | SWOT 导出 + 对象导向呈现 |

## 输出元数据（每次体检带上）
`分析场景 / 日期 / 档位 / 评级`。便于截图、对比、沉淀成内容素材。
