# 覆盖与数据源:三视角 · 源地图 · 业务指标

## 三视角（判断表现与潜力 · 覆盖够不够的第一关）
- **看自己**:目前销量、达成、增长率(GR)
- **看周边**:同类竞品、相关产品、竞争格局
- **看全局**:其他指标(病人数、床位、就诊量、手术量等)
- 潜力分层次:先看量(UNIT)再看额(VALUE);大潜力 vs 小潜力别混着比。
- 体检:只"看自己"没"看周边/全局" = 覆盖不及格。

## 数据源地图（通用类别 · "该拉的源拉全没"检查表）
- **外部·市场**:第三方市场销售数据库(总体市场/自身/竞品规模与趋势)· 处方数据(科室级)· 医院级数据(分级/销量/潜力)· 流行病学/潜力(患病→诊断→治疗漏斗)
- **内部**:销售/达成/增长贡献/生产力 · 销售过程数据(覆盖/拜访/频率/信息传递)· CRM
- **缺失怎么估算**:同类不同种 / 同种竞品 / 相似产品 / 门诊量 / 床位 / 手术量 / 现实观察统计 / 问卷调研

## 业务指标（读得对不对）
- **份额(MS / 相对份额)· 潜力(size)· 增长(GR / CAGR / EV 预期增长)**——每一层(全国→大区→城市→医院/科室)都问这三维看了没。
- PF 漏斗(患病→诊断→治疗→品牌)· 份额三阶(声量→心智→市场)· 销潜比 / 准潜比。

## 挑刺判定
- 只拉内部结果数据(销量),没拉外部潜力/竞品 = 覆盖不及格。
- 缺源不标 `[待确认]`、直接下结论 = 误读风险。
