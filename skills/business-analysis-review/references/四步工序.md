# 四步工序:解读 · 切割 · 关联 · 递进

分析有固定工序,不是看一眼下结论。挑刺时看"工序走到第几步"。

## 解读
看图说话——这组数/这张图在说什么。先总后分。
- 体检:有没有把数据"读成一句话"(而不是只贴表)。

## 切割（细分的切入点,暴露问题）
归类分组,初步暴露问题。常见切割维度:
- **地域**:城市级别(一/二三线/其他)· 主要贡献城市 · 高于/低于平均增长
- **客户/对象**:重点层级 · TOP 金额 / TOP 潜力 / 其他 · 高/低增长
- **结构**:按科室 / 产品线 / 渠道
- 体检:有没有切割?还是一锅端只看总量。

## 关联（同对象多角度并置,暴露深层）
把"同一对象、不同角度的数据"放一起对比,暴露深层(串联多屏)。典型:
- 看自己 + 看周边(竞争格局）
- 销售额 TOP20 + 潜力 TOP20（找错配 = 销潜比断点）
- 城市销售分布 + 城市潜力分布
- 某对象表现 + 标杆(区域最佳）
- 体检:有没有把两组数据"叉起来看"?

## 递进（往深挖,至少 3 层）
"下一本书就在你正看的这本里。" 三问:
1. 为什么会这样?还需要再找哪些数据?暴露了什么深层问题?
2. 我接下来要行动的是什么?
3. **避免单一简单层次:数据到手后至少分析 3 层。**

## 挑刺判定
- 只停在 解读/切割 = 浅层 = 不及格;到 关联 = 良好;**递进到"行动启示" = 优秀**。
- 招牌追问:"这份分析,你挖到第几层了?"
