# 主菜配菜 & 客户参与度

## 主菜配菜
- **主菜 = 产品核心信息(这场要传的那一个关键信息)**;**配菜 = 疾病治疗 / 专业能力 / 个人成长三个角度的"额外收获"内容**(与 `体验公式与三技巧.md` 技巧一同一定义)。
- 体检看:**主菜配菜比例对不对**——配菜喧宾夺主(光热闹没干货)、或主菜太重没有承接(医生消化不了),都扣分。
- **不同类型/规模的活动,主菜配菜配比不一样**(科室会 vs 大型论坛);体检按活动类型判断比例是否合理,用户可在 `EXTEND.md` 配置自己的偏好。

## 客户参与度四级(越往上越值钱)
| 级 | 状态 | 体检含义 |
|---|---|---|
| 1 | **有趣** | 医生觉得好玩/不无聊(最低门槛,只到这级=偏配菜) |
| 2 | **和我有关** | 内容接上医生自己的病人/困境 |
| 3 | **专业升级** | 医生学到能用的新认知/新打法 |
| 4 | **命运共同体** | 医生觉得"这是我们一起在推的事"(最高,沉淀关系资产) |

→ 体检判定:只到 1–2 级 = 一般;到 3 级 = 良好;到 4 级且有资产沉淀 = 优秀。

## 资产视角(配合自检清单最后一问)
好活动**一次办完留下可复用资产**:HCP 网络 / 代表能力提升 / 患者数据 / 可二次传播的内容。只消耗预算不沉淀 = 扣分。
