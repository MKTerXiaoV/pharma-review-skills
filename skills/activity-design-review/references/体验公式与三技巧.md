# 活动质量 · 体验公式 + 三技巧(体检主透镜)

> 源自用户自有活动质量方法(脱敏)。与「五大虚症」互补:五大虚症=挑失败模式;本篇=正向优化杠杆 + 一个净值公式。体检时两套都过。

## 原理(地基):客户体验 = ∑ 正体验 − ∑ 负体验
体检一个活动,先算这道"加减法":正体验够不够、负体验有没有在悄悄扣分。
- **正体验**(价值感/愉悦感/尊重感):学到新知 · 提升技能 · 获得认可 · 有趣的互动
- **负体验**(障碍/麻烦/不愉快/认知负担):会议冗长 · 内容无聊 · 参与度低(单纯听讲)· 付出大量成本(时间/精力/操作复杂)
→ 体检判定:方案里**负体验项**(尤其"单纯听讲""操作复杂")没被消除 = 净值被拖低,扣分。

## 技巧一:主菜 & 配菜合理搭配 —— 目标:提升内容价值
- **主菜 = 产品核心信息**(这场要传的那一个关键信息)
- **配菜 = 三个角度**:① 疾病治疗角度(领域热点)② 专业能力角度(如手术/技术新进展)③ 个人成长角度(如科室管理)
- **设计时问两个问题**:① 这场活动要解决什么问题,核心信息(主菜)是什么?② 有什么其他内容(配菜)是客户听完很有收获的?
→ 体检:主菜含糊/缺失、或全是主菜没配菜(客户没"额外收获")、或配菜喧宾 = 扣分。
（不同活动类型主菜配菜配比不同,用户偏好可在 EXTEND 配置）

## 技巧二:主动设计峰终体验 —— 目标:创造关键时刻
- **核心原理**:对一段经历的评价,主要由**峰值 & 终值**决定(峰终定律)。
- **峰值检查**:活动里安排了哪个环节让客户感到"**被看见**""**被尊重**"、极度愉悦的高光时刻?
- **终值检查**:活动结束有没有给客户**具体真诚的反馈 / 值得回忆或带走的素材**?
→ 体检:平铺无峰值、或结尾是"领导致辞/合影"草草收场 = 扣分。

## 技巧三:散点连成线 —— 目标:沉淀品牌资产
- 从"**分散的点状活动**" → "**可沉淀的活动平台**"。
- **不要让一个活动无缘无故来、无缘无故走。**
- **检查**:① 活动平台有没有**容易传播、有记忆点的名字**,在持续经营?② 一系列活动在客户看来是**逻辑连贯、层层递进**(像连续剧)的吗?
→ 体检:孤立一次性活动、年年重起炉灶、没有系列名/没有递进 = 扣分(没沉淀资产)。
→ 把孤立活动做成**有名字、可持续经营的系列/平台**,是沉淀品牌资产的关键。
