# 活动设计体检 · 评分卡

## 评级(及格 / 良好 / 优秀)
评级只有三档;低于及格线不单设档,表述统一为**"未及格线,建议返工"**。逐维达标度合成:

| 维度 | 未及格线(命中即建议返工) | 良好 | 优秀 |
|---|---|---|---|
| 紧锁策略 | 反推不出策略 | 能反推 | 第一页明写且贯穿 |
| 业务感 | 走出去不知道干啥 | 有"下一个病人这样试" | 落到具体病人特征+动作 |
| 反人性 | 有额外劳动环节 | 无 | 无 + 价值全由主办给 |
| 峰终 | 平铺 | 有峰值 | 峰值绑核心信息 + 终值有承接 |
| 主菜配菜/参与度 | 配菜喧宾/只到"有趣" | 比例合理/到"和我有关" | 到"专业升级/命运共同体"+ 留资产 |

> 及格 = 三条一票否决项全部不命中(策略不脱钩、无反人性、结尾有业务感);良好/优秀按上表逐维抬升。与 SKILL.md「评级(大白话)」一致。

## 一票否决(命中任意一条 = 未及格线,建议返工)
- 有反人性设计(要医生额外劳动)
- 结尾没有任何业务感动作(医生不知道明天怎么做)
- 策略完全脱钩(纯为办而办)

## 输出元数据(每次体检带上)
`活动名/受众 · 体检日期 · 档位(速检/标准/深档) · 评级`。便于截图、对比、沉淀成内容素材。

## 挑刺硬规则
**每条挑刺必引方案原句/原环节**(指到"哪一项议程/哪句话")——这是"实用体检"和"正确的废话"的分界。
