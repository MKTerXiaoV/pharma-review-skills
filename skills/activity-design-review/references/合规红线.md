# 合规红线(发出去前必清)

## 体检对象(活动方案)常见合规雷
- **夸大疗效 / 绝对化**:活动宣传语里的"最佳实践""唯一选择""根治"等 → 标出来。
- **诱导超适应症**:议程/病例引导未获批人群或用法 → 标出来。
- **诱导性福利**:与处方挂钩的礼品/费用 → 红线,体检必拦。
- **未审材料**:活动用的数据/幻灯/病例须经 MA/RA 审核;体检结尾固定提示。

## skill 自身红线
- **不编**:不确定的获批/指南/数据 → 标 `[出处占位]` 或提示核实,不猜。
- **只体检不代设计**:不替用户写日程/活动方案(生成类让出)。
- **脱敏**:样张用虚构产品(慢病/降压),避真实管线;不暴露用户身份/雇主。

## 固定收尾
每份体检结尾带:`本体检为方法校准,具体活动材料须经 MA/RA 审核后执行。`
