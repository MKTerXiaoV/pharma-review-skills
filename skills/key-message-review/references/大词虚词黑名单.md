# 大词虚词黑名单

**判定原则：把产品名换成任意竞品，这句话还成立 → 它是空的。**

医药市场内容里最常见的三类"空话"，扫到就标红，并追问"它其实想说什么"。

## A. 万能正确语（对，但推不出任何动作）

| 词 / 句式 | 为什么空 | 追问 |
|---|---|---|
| 全面提升 / 全面覆盖 | "全面"=没有重点 | 提升谁对哪句话的认知？ |
| 深度赋能 / 多维度赋能 | 黑话，无实指 | 给医生一句话还是一个工具？ |
| 创新引领 / 引领新标准 | 自我加冕 | 在哪一点上领先？一句话 |
| 打造差异化 | 差异没说 = 没差异 | 比谁、对哪群病人、强在哪 |
| 助力 / 携手 / 共筑 | 口号填充 | 删 |

## B. 绝对化 / 最佳类（同时是合规雷，见 合规红线.md）

卓越 · 最佳 · 彻底 · 根治 · 唯一 · 理想之选 · 完美 · 100%
→ **一律拦截**。改成可验证、有边界的表述。

## C. 空泛抒情（听着热血，医生不信）

新时代 · 新篇章 · 全新希望 · 开启 · 久经考验 · 重磅
→ 删，或换成具体的病人场景与一句能记住的话。

## 改写方向（不是删干净，是落地）

空话的反面是**三个具体**：
1. **具体的病人**：哪一群（不是"广大患者"）。
2. **具体的对比**：比谁、强在哪一点。
3. **具体的动作**：医生看完明天会做什么。

> 三个具体齐了，这句话就只有你的产品能说。
