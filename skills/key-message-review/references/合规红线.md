# 合规红线（处方药内容初筛）

> 本清单是**常见风险初筛**，不替代贵司医学/法规审核（MA/RA）。产出前必过；命中即先拦截、再改写。

## 1. 绝对化 / 最佳类宣称 —— 拦

卓越 · 最佳 · 最优 · 彻底 · 根治 · 治愈 · 唯一 · 完美 · 100% · 理想之选 · 首选（无依据时）
→ 改成有边界、可验证的表述（如"在 X 人群中，相比 Y……"）。

## 2. 裸数据（无出处的疗效/安全数字）—— 拦

任何"显著延长 / 大幅降低 / 有效率 XX%"若无来源：
→ 标注 `[数据 + 出处占位]`（试验名 / PI 页码 / 文献），**绝不杜撰数字**。

## 2b. 存疑就查，不猜不编 —— 触发核实

涉及**外部可变事实**(竞品数据/获批状态/指南纳入/发布时间等)且不确定时：
→ **触发 web 核实**(走 web-access)取一手来源，或标 `[待核实]` / `[出处占位]`；**绝不凭印象断言**。知识有截止日，"最新获批/最新版指南"尤其先查。

## 3. 超适应症 / 暗示扩适应症 —— 拦

- 只在用户声明的**核准适应症**内表达。
- 警惕把"轻中度原发性高血压"模糊成"高血压患者"这类悄悄放大边界的写法（重度、继发性其实都不在核准范围内）。

## 4. 贬低竞品越界 —— 拦

可做基于证据的比较，不做攻击性/无依据贬低。

## 5. 安全性"只报喜" —— 提示

回避已知不良反应、把"安全"讲成"无副作用"=信任反噬且不合规。
→ 坦诚已知不良反应，讲成"可预期、可管理"。

## 6. 处方诱导 / 面向公众 —— 拦

不写诱导处方话术；不生成面向公众的处方药广告内容。

## 7. 用户禁用词（来自 EXTEND.md）—— 逐条拦

读取 `EXTEND.md` 的禁用词与术语规范，命中即改写（例如不规范的疾病/治疗线术语）。

## 拦截后的标准动作

1. 指出命中的是哪一条红线。
2. 给出合规改写版。
3. 在产出末尾附一句：**"数据以说明书/临床试验为准；对外材料须经 MA/RA 审核。"**
