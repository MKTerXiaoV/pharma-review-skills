# 越用越准:约束型迭代法(私域护城河)

> 来源:baoyu《图解Skill》ch4「写作风格技能」招牌机制 + baoyu skill「输出即配置」惯例。
> 关键信息体检是**约束型**(定义"好 key message"的判断标准)。约束型最强的进化方式 = **把你真实改过的 message 喂回来,让标准越来越贴你的判断**。别人抄得走大词黑名单,抄不走"你改过几百句 message 后的手感"。

## 四步循环(每改一句真实 message 就做一次)
1. **体检**:用本 skill 体检一句真实 key message(**先脱敏**:虚构产品名,去内部数据)。
2. **你自己动手改**:把它**亲手改成你满意的样子**——不是说"这里改一下",是真改。你真正在意的标准,改的时候才暴露。
3. **让 AI 提炼规则**:把"改前 + 你的改后"一起发给 AI,问:"我删了哪些词、为什么、背后是什么可复用规则?"(常得出:新的禁用词 / 新的"有信息量"判据)
4. **更新技能**:让 AI 把新规则写进 `EXTEND.md`(你的私有偏好)或 `references/大词虚词黑名单.md`、`有效信息三要素.md`。

几轮下来,体检的挑刺会越来越像你本人——**护城河:它学到的是你的判断**。

## 注意(book ch4 两个坑)
- **脱敏**:改过的真实 message 也要脱敏再喂(去真实产品名/公司/内部数据);沉淀进公开 references 的尤其要清。
- **黑名单别堆太长**:禁用词 ≤ 10 条最核心的;长了会"打架"。定期让 AI **归纳合并**成更少的规律。

## 通用机制
这是**所有体检系列 skill 通用**的自我进化回路(关键信息体检 / 活动设计体检 / 未来的体检都适用)。详见事业文档 `skill制作SOP.md`。
