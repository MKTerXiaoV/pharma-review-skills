# 有效信息三要素

一条关键信息（key message）"实不实"，看它是否同时具备三要素。缺一条就偏虚。

## 三要素

1. **具体的病人** — 指向一群可识别的病人，不是"广大患者/所有人"。
   - 弱：高血压患者　→　强：清晨血压控制不佳、多药联用依从差的病人
2. **可记住的一句话** — 医生能原样复述、甚至带进诊室对病人说。
   - 弱：平稳降压　→　强：「一片，管住清晨那一峰」
3. **医生明天的动作** — 看完这条，医生会改变某个具体决定。
   - 弱：提升认知　→　强：对这群病人优先考虑单片复方

## 信息结构：1 主信息 + 支撑，不是一堆平级口号

- ❌ "机制突破 / 疗效升维 / 安全革新"——三个并列 = 信息堆砌，医生一个都记不住。
- ✅ **1 条主信息**（最硬的差异化）+ **2 条支撑**（疗效、安全各一），每条都过三要素。

## 自检三问

- 这句话换成竞品名还成立吗？（成立 → 空）
- 医生能原样复述吗？（不能 → 没记忆点）
- 看完他明天会做什么？（说不出 → 没动作）
