# 视觉权重 vs 信息权重 倒挂检查

适用：展架 / 易拉宝 / 海报 / slide 等**有版面**的物料（不只纯文字 message）。

## 核心病

把**最不能区分你**的大词口号做成最大的字，
把**只有你能说**的差异化数据埋进脚注。
→ 视觉权重 ↔ 信息权重 **倒挂**。

## 检测四步

1. 找出物料上**字最大的 3 个元素**——多半是口号（强效出击 / 革故鼎新 / 疗效卓越 / 创新性）。
2. 把产品名换成竞品，这 3 句还成立吗？**成立 = 玻璃珠**（人人都有）。
3. 找出**字最小的脚注**——多半是真钻石：特定人群优势、头对头、指南纳入、具体数字（如"BRCAwt mPFS 25.5 个月""纳入 2025 指南"）。
4. 若"大字=玻璃珠、小字=钻石"成立 → **倒挂确诊**。

## 修法

- **钻石上头版，玻璃珠降级或删。**
- 视觉层级 = 信息层级：最值钱的信息，字最大。
- 意境/抒情词最多留 1 个做视觉氛围，**不当信息用**。

## 一句话

> 医生在展位前停留 3 秒，3 秒只看大字。**大字每家都一样 = 这 3 秒白给了。**

## 实战印证

2026-06 三家妇科肿瘤竞品展架同时犯此病：大字全是"强效出击/革故鼎新/疗效卓越"（可互换），而 BRCAwt 25.5 个月、中华医学会指南纳入这类硬差异化全在脚注。详见验收记录。
