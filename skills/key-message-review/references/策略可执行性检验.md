# 策略可执行性检验（主谓宾还原法）

判断"定位/策略是否优秀"的核心方法：**能不能还原成一句有主谓宾的可执行话。**

## 检验句式

把定位/策略填进这个句子，填得满 = 实，填不满 = 虚：

> 我们要 **[抢下 / 守住哪一群病人]**，
> 靠的是 **[本产品 vs 哪个竞品，在这群病人身上的哪一个具体优势]**，
> 让医生 **[改变哪个具体处方动作]**。

三个空都能填具体 → 良好/优秀。
一个填不上 → 及格或不及格。

## 常见"虚"的形态

| 写法 | 病症 | 诊断 |
|---|---|---|
| "成为 X 领域领导品牌" | 把**结果**当策略 | 领导品牌是奖杯，不是打法 |
| "提升市场份额/处方量" | 把**目标**当策略 | 目标不告诉销售明天做什么 |
| "学术会+科室会+患教+数字化" | 把**菜单**当菜谱 | 四样都做=没选；问"砍两个留哪两个" |
| "疗效好、安全好、方便" | 自我表扬 | 没有对手、没有病人 = 没有差异化 |

## 一个狠问题（用来戳穿"虚"）

> "你列了这些举措，**砍掉一半，你留哪一半？为什么？**"
> —— 答不出"砍哪个"，就是还没有策略。

## 与"主攻方向"配套

定位选定后，要能指出**主攻一个点**，而不是平均用力。把最硬的差异化当矛尖，其余作配合。
